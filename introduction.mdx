# G5API

G5API is an Express-based REST API backend for Counter-Strike 2 (CS2) competitive match management. It serves as a replacement for the legacy get5-webpanel, providing a modern Node.js/TypeScript foundation that integrates with the [MatchZy](https://shobhit-pathak.github.io/MatchZy/) and [PugSharp](https://github.com/Lan2Play/PugSharp) CS2 plugins.

## What G5API does

G5API allows you to create, manage, and control CS2 matches programmatically. It integrates with game servers via RCON and provides callbacks for plugin events.

## Supported plugins

G5API supports match event callbacks from:

- **[MatchZy](https://shobhit-pathak.github.io/MatchZy/)** — A CS2 practice and match plugin
- **[PugSharp](https://github.com/Lan2Play/PugSharp)** — A CS2 pug/match management plugin
- **[get5](https://github.com/splewis/get5)** — The original CS:GO match plugin (legacy support via `/match` routes)

## Front-end

G5API is backend-only. For a ready-made front-end, use **[G5V](https://github.com/phlexplexico/g5v)**, which provides a Vue-based UI that works out of the box with G5API.
