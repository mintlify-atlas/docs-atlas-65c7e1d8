---
title: "List Seasons"
description: "Retrieve all seasons from the G5API system."
---

## Overview

Returns all seasons recorded in the system, including each season's default CVARs if configured. This endpoint is publicly accessible — no authentication required.

## Endpoint

```
GET /seasons/
```

## Response

<ResponseField name="seasons" type="array">
  Array of season objects.

  <Expandable title="Season object fields">
    <ResponseField name="id" type="integer">
      The unique season ID.
    </ResponseField>
    <ResponseField name="user_id" type="integer">
      The ID of the user who created the season.
    </ResponseField>
    <ResponseField name="name" type="string">
      The name of the season.
    </ResponseField>
    <ResponseField name="start_date" type="string">
      ISO 8601 start date for the season.
    </ResponseField>
    <ResponseField name="end_date" type="string">
      ISO 8601 end date for the season. `null` if the season has no scheduled end.
    </ResponseField>
    <ResponseField name="cvars" type="object">
      Key-value pairs of default server CVARs applied when this season is selected. Omitted if no CVARs are configured.
    </ResponseField>
  </Expandable>
</ResponseField>

## Example

```bash
curl https://your-g5api.example.com/seasons/
```

## Error Responses

| Status | Description |
|--------|-------------|
| `404`  | No seasons found in the system. |
| `500`  | Internal server error. |
